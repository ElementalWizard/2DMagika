2D-Magika

A programming language for Spanish speakers, designed to help those who have dificulty grasping programming or English in general. It can be considered especially helpful for children who may not understand fully by just using single keywords. It is a basic grammer consisting of variables, conditionals, and loops to give a basic understanding of skills that can translate to other programming language by defining keywords as sentences in natural language.

Java

For this project you will need to have the JDK installed and in your path. You can find the JDK here

To verify if java is installed and in your system path open a terminal and type

$ java -version

You should receive the version in the next line.
